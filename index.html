White paper for a new layer 1

It is a dapp chain

Vision

Suwappu is a new dapp chain that will be focusing on simply aggregating, swapping, lending, building the largest stablecoin pools.

  • Liquidity Fragmentation is increasing across chains
  • Bridge risk causes uncertainty with wrapping and unwrapping
  • Complex UI/Managing securing wallets diffuclt
  • High latency and network congestion
  • Limited cross chain functionality

Building on top of the progress of defi primitives like UNI COMP AAVE and CURVE we a working on something new

Suwappu is designed to overcome these limitations by introducing:

  • An innovative liquidity aggregation mechanism
  • A move chain solution for high speed, low cost, and parralization
  • Intuitive user interface
  • Cross chain interoperability and modularity
  • Entangled Rollups + zk Proofs enable cross chain swaps natively securely and bridgeless

Technical Architecture

Smart Contract Layer

Suwappu core functionality is implemented through smart contracts that will handle the Order Matching, Liquidity pool managment, Token Swaps, and Governance Mechanisms

Dapp Chain Scaling Solution

To address calability issues, Suwappu uses ZKM's Entangeled Rollup plus the MoveVM

  • High transaction throughput by parralizaiton
  • Consensus will uses SUI moves sharkpool model
  • ZKM's money managers will act as channels for cross chain swaps enabling a bridgless and trustless model for native liquidity
  • Native account abstraction from protocol level will enable seedless and passphrase wallets while eliminating gas $USDC will be the native gas token

Liquidity Aggregation

Suwappu will introduce a novel liquidity aggregation protocol that

  • Connect thru 1inch for EVM Liquidity
  • Build connections thru non EVM liquidity first SOL,APTOS,SUI,Movement
  • Enable dark pools to minimize slippage for large trades

Cross-Chain Bridgeless Swaps

Using ZK proofs, we can enable a novel interoperability model between blockchain networks regardless layer 1,2s, or EVM,SVM,MVM,etc
